# TS Blog Aggregator

The TS Blog Aggregator (or gator) is a CLI tool to fetch and manage RSS feeds. This is a small project of the boot.dev programming courses.

## Commands

- register <user_name>
- login <user_name>
- reset
- users
- agg <aggregation_duration_string>
- addfeed <feed_name> <feed_url>
- feeds
- follow <feed_url>
- unfollow <feed_url>
- following
- browse
